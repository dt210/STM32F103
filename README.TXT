三个key设为外部中断处理紧急事件
串口一用于上位机通信

控制板返回锁状态信息给单片机，单片机发数据给上位机，将锁状态可视化处理
OLED用于显示关键信息

上位机与单片机的通信指令有
开锁指令，关锁指令，锁状态指令
